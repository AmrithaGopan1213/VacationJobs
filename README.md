🏖️ Vacation Job Portal 🌟
My First Step Into the Web Development World 🌐 — Built with Django!

🚀 About the Project
Vacation Job Portal is a web application that connects students looking for temporary jobs during holidays with companies offering short-term roles. This was the first real-world web application I built using Django, and it marks the beginning of my full-stack journey.
------------------------------------------------------------------------------------------------------------------------------------------------------------------
✨ Installation
git clone https://github.com/denmarkdeni/Vacationjob.git
cd Vacationjob
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
--------------------------------------------------------------------------------
✨ Features
🔐 User Registration & Login
🧑‍🎓 Student Dashboard – View and apply to job listings
🏢 Company Dashboard – Post and manage job opportunities
📃 Job Listings – Search, apply, and track job applications
📬 Admin Panel – Manage all users and posted jobs
